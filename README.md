# Happy New Year 2026

## 프로젝트 개요

지인들에게 새해 인사 편지를 링크로 전달하는 모바일 최적화 웹 애플리케이션입니다.
프론트엔드 전용 프로젝트로 AWS S3 + CloudFront를 통해 배포됩니다.

## 동작 방식

수신자의 이름과 편지 내용을 querystring으로 전달합니다.

- JSON 데이터를 base64로 인코딩하여 URL 파라미터(`f`)로 전달
- 예시: `https://happynewyear-2026.whitekiwi.link/?f=eyJuYW1lIjoi7ZmN6ri464uZIiwibWVzc2FnZSI6Iuyg...`

```json
{
  "receiverName": "김토스",
  "receiverLabel": "님께",
  "message": "편지 내용...\n.......",
  "senderName": "장키위",
  "senderLabel": "올림",
  "senderPhone": "01012345678"
}
```

## 기능 요구사항

### 편지 보기 (메인 페이지)

1. 링크로 접속하면 개봉되지 않은 편지가 보여야 함
2. 편지를 클릭하면 다음 순서로 애니메이션 실행:
   - [1] 편지 봉투가 열리고
   - [2] 접힌 편지가 안에서 튀어나오고
   - [3] 편지가 펼쳐지면서 내용이 보여짐
3. 편지에는 수신자의 이름과 내용, 발신인이 적혀 있어야 함
4. 편지 아래에는 "용돈 보내기" 버튼과 "답신 보내기" 버튼이 있음
5. "용돈 보내기" 버튼 클릭 시:
   - "용돈은 마음만 받을게요. 밥 사주세요!" 팝업이 나타남
   - 팝업에는 "밥사주기" 버튼이 있음
   - "밥사주기" 버튼 클릭 시:
     * 전화번호가 클립보드에 복사
     * "전화번호가 복사되었어요" 알림 표시 (3초간)
     * 발신인 전화번호로 전화 연결 (`tel:` 스킴)
6. "답신 보내기" 버튼 클릭 시:
   - 편지 작성 페이지(`/kiwi`)로 이동
   - 수신자와 발신자가 바뀌어서 미리 채워짐 (답신 작성 용이)

### 편지 작성 (`/kiwi` 경로)

7. `/kiwi` 경로로 접근하면 편지 작성 창이 나타남
8. 작성 폼에서 다음 항목을 입력:
   - 수신자 이름 (receiverName)
   - 수신자 표기 (receiverLabel)
   - 편지 내용 (message)
   - 발신인 이름 (senderName)
   - 발신인 표기 (senderLabel)
   - 전화번호 (senderPhone)
9. 입력 완료 후 "링크 복사" 버튼을 누르면 base64로 인코딩된 URL이 클립보드에 복사됨
   - 예: `https://happynewyear-2026.whitekiwi.link/?f=eyJyZWNlaXZlckxpbmUi...`
10. 모든 편지는 코트라 손글씨체로 표시됨
